HTML Y PYTHON

HTML (HyperText Markup Language) es un lenguaje de marcado que se utiliza para estructurar y mostrar páginas web. Es el componente básico de la web y se utiliza para definir el contenido y la estructura de una página web

Python es un lenguaje de programación de alto nivel, interpretado y de código abierto, conocido por su sintaxis clara y legible, que se asemeja al lenguaje natural, lo que facilita su aprendizaje y uso
